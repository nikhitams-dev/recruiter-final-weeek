P# 🇮🇳 AI Recruitment Production Suite (Track A)

This is a production-ready recruitment assistance application designed to automate resume screening and candidate ranking.

## 🚀 Live Demo
**Public URL:** https://f3beed2ac646fe57dc.gradio.live/

## ✨ Features (Track A Checklist)
- **Dashboard:** Interactive candidate pipeline view.
- **ATS Scoring:** Automated ranking using NLP & Cosine Similarity.
- **Batch Processing:** Handles multiple resume uploads.
- **Hiring Analytics:** Visual bar charts for candidate comparison.
- **Export Functionality:** One-click CSV report generation.

## 🛠️ Setup & Installation
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
